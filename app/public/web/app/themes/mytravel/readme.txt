MyTravel is the perfect theme for your Travel agency, Hotel aggregators and Tour consultancy website.

== Description ==

MyTravel is a complete Travel WooCommerce Theme for hotels, vacations, rentals, cars, yachts and activities. It integrates with the WooCommerce Easy Booking (free) and WooCommerce Bookings (paid) bookings plugin to provide you with the booking features require for managin hotels, vacations, rentals, cars, yachts and activities.

MyTravel theme is built with WooCommerce. The core features such as hotels, vacations, rentals, cars, yachts and activities are built as WooCommerce products. MAS Travels plugin integrates with WooCommerce Bookings and WooCommerce Easy Bookings plugin. It also adds a custom taxonomy called Product Format. Product Format decides how a product is displayed and what are its custom fields. The custom fields are provided ACF plugin.

Building MyTravel with WooCommerce allows you to make best use of advanced features of WooCommerce such as built-in payment processing from leading providers, manage orders and customers, optimize convertions, etc.

For more information about MyTravel please go to https://themeforest.net/user/madrasthemes/portfolio

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on 'Upload Theme' button and upload the theme package that is downloaded from ThemeForest.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://docs.madrasthemes.com/mytravel/ guides on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
